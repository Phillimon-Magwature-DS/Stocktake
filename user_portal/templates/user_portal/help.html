Staff Portal Help

Instructions for using the stocktake system

Getting Started

  1. Select your department from the Home page
  2. Get an access code from your HOD for the current stocktake
  3. Enter the access code to start recording counts

Recording Counts

  • Update: Replaces the existing count with your new numbers
  • Add: Adds your numbers to the existing count (use when multiple people are counting)
  • Enter expiry dates in YYYY-MM-DD format when known

Viewing Data

  • Use the "View Tables" page to see all stocktake data for your department
  • Export to Excel or CSV for further analysis
  • The progress indicator shows how much of the stocktake is complete

Troubleshooting

  • Can't access a stocktake? Verify the access code with your HOD
  • Wrong department? Use "Switch Department" to change
  • Data not saving? Check your internet connection and try again
