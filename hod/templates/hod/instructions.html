System Instructions

Guide to using the Hospital Stocktake System

Getting Started

  1. Import your department's drug list using the "Import Drugs" function
  2. Create a new stocktake table when you need to record inventory
  3. Share the access code with staff who will be counting inventory

Managing Stocktakes

  • Each stocktake creates a snapshot of your inventory at a point in time
  • View previous stocktakes in the Reports section
  • Export data to Excel or CSV for further analysis

Analyzing Data

  • Use the Reports section to view trends over time
  • Visualize drug usage patterns with the interactive charts
  • Identify drugs that need reordering based on usage trends

Tips

  • Perform regular stocktakes for accurate data
  • Use consistent naming for stocktake tables (e.g., "Monthly - June 2023")
  • Review expiry dates regularly to minimize waste
